# Atualização V20 - Orçamentos

## O que mudou
- Adicionada opção de editar orçamento.
- Adicionada opção de excluir orçamento.
- Se o orçamento aprovado for editado, o item criado no Plano/Ficha também é atualizado.
- Se o orçamento aprovado for excluído, o item criado no Plano/Ficha também é removido.
- O botão "Aprovar" foi renomeado para "Aprovar para plano" para evitar confusão com o financeiro.
- Adicionado aviso explicando que aprovar orçamento não lança cobrança no financeiro.

## Explicação
Aprovar orçamento serve para transformar a proposta em plano de tratamento do paciente. Para lançar cobrança/recebimento, deve usar "Nova cobrança no financeiro".
