# Atualização V25 - Primeiro acesso realmente aparece

## O que mudou
Agora, se o sistema estiver com apenas o usuário padrão `admin`, ele abre direto a tela de primeiro acesso.

A clínica escolhe:
- Usuário de entrada
- Senha de entrada
- Senha do financeiro

Depois de salvar:
- O usuário padrão `admin` é substituído pelo usuário escolhido
- A senha do financeiro é salva nas configurações
- O sistema não mostra mais a tela de primeiro acesso

## Importante
Não precisa apagar banco nem perder dados.
