# Atualização V23 - Marcar como pago no financeiro

## O que mudou
- O botão "Baixar" foi renomeado para "Marcar pago".
- Agora o financeiro deixa mais claro quando o lançamento é pago total e quando é pagamento parcial.
- Para lançamentos pendentes aparece:
  - Forma de pagamento
  - Marcar pago
  - Pagamento parcial
- Para lançamentos já pagos aparece:
  - Ver pagamentos

## Observação
A função já existia como "Baixar", mas o nome podia confundir. Agora está explícito.
